# Babylon Game — правила разработки

## Золотое правило: фича → энциклопедия → перевод

**Любое внедрение, затрагивающее игровой опыт (новая механика, система, экран,
существенное изменение баланса), обязано сопровождаться:**

1. **Описанием в энциклопедии.** Энциклопедия — это `buildEncyclopedia()` в
   `js/encyclopedia.js`, отображается в разделе **Академия** (`viewAcademy`,
   `js/ui.js`). Добавь/обнови соответствующий раздел. Вся проза проходит через
   `t(...)`, данные (имена/описания) — тоже через `t(...)`; статичные списки по
   возможности строй из игровых данных (`CLAN_UPGRADES`, `SPELLS`, `WORLDS` …),
   чтобы текст не расходился с механикой.
2. **Переводом на ВСЕ языки.** Новые русские строки-ключи добавляются в
   английский эталон (`I18N.en` в `js/i18n.js`) и в каждый языковой пакет
   `js/lang/<код>.js` (es, de, fr, pt, it, zh, ja, ko, th). Процесс и правила —
   в `docs/LOCALIZATION.md`.

Фича считается завершённой, только когда выполнены оба пункта. Не оставляй
непереведённых ключей и недокументированных механик.

## Контроль перед деплоем

- `node check-i18n.js <код>` для каждого языка — «не хватает 0» (наборы ключей
  совпадают с эталоном `en`).
- `node scan-keys.js <код>` — нет ключей в коде, которых нет в словаре.
- `node tools/i18n-export.js` — пересобрать `docs/i18n.reference.tsv`,
  `docs/translation-keys.md`, `docs/i18n.skeleton.json` после изменения ключей.
- `node --check` для изменённых `.js`.

## Деплой

- Фронтенд (Cloudflare Pages, проект `babylon`): поднять `?v=NN` у всех
  `<script>`/`<link>` в `index.html`, затем `wrangler pages deploy` (включать
  `index.html admin.html css js docs img`).
- Воркер (`worker/save-worker.js`, `babylon-save`): деплоится отдельно
  (`cd worker && npx wrangler deploy`). Серверные изменения деплоит владелец.

## Локализация — кратко

Источник ключей — русский. См. `docs/LOCALIZATION.md` для полного гайда.
Сохранять дословно: плейсхолдеры `{...}`, эмодзи, числа, символы `«» × → −`,
теги `<b>…</b>`. Не переводить: имена игроков, названия кланов, пользовательские
названия сборок/питомцев. Имена рейд-боссов — переводятся.
