# Папка артов «Вавилон»

Сюда складываются PNG-картинки. Имена файлов и описания — в `../ART_PROMPTS.md`.
Движок сам подхватывает файл, если он есть; если файла нет — рисуется процедурная SVG-заглушка.

- `worlds/` — фоны 12 миров: `world_NN_<слаг>.png`
- `mobs/`   — существа: `<слаг>.png` (например `lyucifer.png`, `chernyy_drakon.png`)
- `items/`  — предметы: `weapon_<слаг>.png`, `armor_<слаг>.png`, `jewelry_<слаг>.png`, `potion_<слаг>.png`
- `tower/`  — `banner.png` и здания: `<слаг>.png` (например `arena.png`, `magazin.png`)
